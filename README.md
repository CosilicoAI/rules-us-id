# rules-us-id

Idaho tax and benefit rules encoded in Cosilico DSL.

## Structure

```
rules-us-id/
├── statutes/       # Idaho Code encoded rules
└── regulations/    # Idaho Administrative Code (IDAPA) rules
```

## Overview

This repository contains machine-executable encodings of Idaho state tax and benefit law, including:

- **Idaho Code Title 63** - Revenue and Taxation
- **IDAPA 35** - State Tax Commission Rules

## Usage

Rules in this repository are designed to be loaded by the Cosilico runtime for microsimulation and policy analysis.

## Sources

- [Idaho Legislature - Idaho Statutes](https://legislature.idaho.gov/statutesrules/idstat/)
- [Idaho Administrative Code (IDAPA)](https://adminrules.idaho.gov/)
